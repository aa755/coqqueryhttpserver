• To locate an object ob (a Coq Definition/Lemma/Inductive e.t.c.), query "Locate ob.". For example, type "Locate rule_true." (excluding quotes) above and click Query.
  • To view the definition of an object, your query should be "Print ob_fullname.", where ob_fullname is is full name of the object ob (including the prefixes).
