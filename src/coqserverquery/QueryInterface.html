• To locate an object ob (Coq definition/lemma/inductive e.t.c), query "Locate ob.". For example, type "Locate rule_true." above and click Query.
  • To view the definition of an object, your query should be "Print ob_short.", where ob_short is the shorter name printed by the locate query on ob. Usually, ob_short is the same as ob. Try the query "Print Opid."
